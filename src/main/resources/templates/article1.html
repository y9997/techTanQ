この流れで進めていけば、記事を読み終えるころには 「Eclipseを使ってJavaのプログラムを動かす」までの基本操作が自然と身につきます。
ぜひ、気軽に手を動かしながら読んでみてください！

Eclipseとは

Eclipseとは、もともとIBMが開発した統合開発環境(IDE)です。

IDEとは、コードを書くためのエディタ、プログラムを動かすためのコンパイラ、バグなどを見つけるためのデバッガなど開発に必要なツールをひとつにまとめたソフトウェアのことです。
IDEを使えばそれぞれツールを用意する必要がなく、簡単に開発環境を整えることができます。

Javaの開発に使えるIDEは他にもIntelliJやVSCode、NetBeansなどがありますが、Eclipseの最大の特徴はオープンソースであることです。
オープンソースなので無料で使用でき、さらにプラグイン(拡張機能)が豊富なのでカスタマイズ性に優れています。

初心者から上級者まで幅広く支持されており、教育機関や開発現場でもよく利用されています。

インストール方法

この章では実際にEclipseをインストールする方法を紹介します。
Eclipseのインストールには、主に次の2つの方法があります。

1.Eclipse公式サイトからインストールする
この場合、入手できるのは英語版のEclipse本体のみです。
日本語で使用したい場合、別途でPleiades(日本語化プラグイン)を追加で導入する必要があります。

2.Pleiades All in Oneをインストールする
Eclipse + Pleiades(日本語化プラグイン) + 便利な開発用プラグイン　のセットになっになったオールインワンパッケージです。
最初から日本語化されており、設定もシンプルなので初心者におすすめの方法です。

本記事では2の「Pleiades All in One」を使ったインストール手順を解説していきます。

macOSの場合

それではEclipseをインストールしていきましょう。

1. Pleiades All in Oneのダウンロード
以下のダウンロードサイトにアクセスします。
Pleiades All in One ダウンロードページ

2. バージョンの選択
任意のバージョンを選択し、ボタンをクリックします。
特にこだわりがなければ最新版がおすすめです。


3. OSと言語の選択
開発対象の言語を「Java」、OSを「Mac」と選択し、「Download」ボタンをクリックします。


4. ファイルを開く
ダウンロードしたdmpファイルを開き、EclipseアイコンをApplicationsにドラッグ&ドロップします。


5. Eclipseの起動
ApplicationsからEclipseを起動します。
警告が表示されたら、OKを選択します。
アプリケーションの実行を許可するため、
「システム環境設定」→「セキュリティとプライバシー」を開きます。
「ダウンロードしたアプリケーションの実行許可」から、「このまま開く」を選択します。



その後、再度Eclipseを起動してください。

6. ワークスペースの設定
workspaceのディレクトリー選択し、「起動」しましょう。
ちなみにworkspaceとは、ソースコードのファイルなどを保管する作業場のようなものです。
ディレクトリーとはその作業場の配置場所のことです。
本記事では初期設定のままで起動しておりますが、任意の場所を指定することもできます。


しばらく待つとeclipseが起動します。


これでEclipseのインストール完了です。次は実際にプログラムを書いてみましょう！

windowsの場合

それではWindows環境でEclipseをインストールしていきましょう。

1. Pleiades All in Oneのダウンロード
以下のサイトにアクセスします。
Pleiades All in One ダウンロードページ

2. バージョンの選択
任意のバージョンを選択し、ボタンをクリックします。
特にこだわりがなければ最新バージョンがおすすめです。


3. OSと言語の選択
開発対象の言語を「Java」、OSを「Windows x64」と選択し、「Download」ボタンをクリック。


4. ファイルのダウンロード
exeファイルのURLが表示されるのでクリックしてダウンロード


5. zipファイルの解凍
ダウンロードしたzipファイルをダブルクリックして開きます。


作成先フォルダのパスを指定して「解凍」ボタンをクリック。
パスが長いと正常動作しないことがあるため、デフォルトのままがおすすめです。


6. Eclipseの起動
解凍が終わったら、/pleiades/2025-09/eclipse(例)から「eclipse.exe」をダブルクリック。


7. ワークスペースの設定
workspaceのディレクトリー選択し、「起動」しましょう。
ちなみにworkspaceとは、ソースコードのファイルなどを保管する作業場のようなものです。
ディレクトリーとはその作業場の配置場所のことです。
本記事では初期設定のままで起動しておりますが、任意の場所を指定することもできます。


しばらく待つとeclipseが起動します。


これでEclipseのインストール完了です。次は実際にプログラムを書いてみましょう！

操作方法

Hello worldの表示

いよいよEclipseを使って、実際にJavaのプログラムを動かしてみましょう！
この章では、最初の一歩として「Hello World!」をコンソールに表示するプログラムを作成して実行します。
短いプログラムですが、Eclipseの基本操作をつかむことができます。
一緒に動かしながら慣れていきましょう！

1.Javaプロジェクトを新規作成する

まず、Eclipseを起動し、新しいプロジェクトを作成します。
左上のアイコンから、またはパッケージ・エクスプローラー内で右クリックし、「新規 → Javaプロジェクト」を選択します。



プロジェクト名を入力し、「完了」をクリックします。

パッケージエクスプローラー内に作成したJavaプロジェクトが表示されたら成功です。

2.クラスファイルを作成する

次に、Javaのソースコードを記述するクラスファイルを作成します。

作成したプロジェクトフォルダを右クリックし、「クラス」を選択します。

クラスファイル名を入力し、「public static void main(String[] args)」にチェックを入れ、「完了」をクリックします。
このチェックを入れることで、mainメソッド（プログラムの起点）が自動生成されます。

エディタ上にクラスファイルが作成され、mainメソッドが自動で記述されます。

mainメソッド内（6行目あたり）に、次のコードを入力します。

System.out.println("Hello,World!");

この命令文は、カッコ内に書かれた文字列や変数をコンソールに出力するものです。

入力が終わったら、上部の保存アイコンをクリックするか、⌘+S（Mac）／Ctrl+S（Windows）で保存します。

3.プログラムを実行する

それでは、作成したプログラムを実行してみましょう。
プロジェクトフォルダを右クリックし、「実行 → Javaアプリケーション」を選択します。

コンソール上に「Hello,World!」と表示されれば成功です！
