【IDE】Eclipseの基本

0.はじめに

この記事はEclipseを知らない、触ったことがない人でも読み終わる頃にはなんとなくわかる、使えることができるよう書いています。
流れとしては、 簡単な概要を学ぶ → Eclipseをインストール → 実際に使ってみる　となっています。
それでは、まずは概要から学んでいきましょう！

1.Eclipseとは

IBM社によって開発された統合開発環境(IDE)の一種です。IDEとは、コードエディタ、コンパイラ、デバッガなどソフトウェア開発に必要なツールをまとめたものです。
Eclipseは充実した基本機能に加え、拡張性やカスタマイズ性に優れているという特徴があり、自分のニーズに合わせた環境を構築することができます。
そのため、初心者から上級者まで幅広く使いやすいIDEとなっています。

2.インストールしてみよう

2.1.インストール方法について

Eclipseのインストール方法は大きく分けて２つあります。 1.Eclipse公式サイトからインストールする。
→この場合、Eclipse単体なので英語版です。日本語で使用したい場合、別途でPleiadesをプラグインする必要があります。

2.Pleiades All in Oneをインストールする。
→Eclipse + Pleiades + その他便利なプラグイン　のセットになっており、初心者におすすめの方法です。本記事ではこの方法を解説します。

2.2.本記事の留意事項

本記事は以下画像のスペックでインストールを行ってます。
バージョンやOS等が異なる場合などは適宜自身のスペックに合わせてください。

それでは、Eclipseをインストールしていきましょう！

2.3.サイトにアクセスする

以下のサイトにアクセスしてください。
https://willbrains.jp/

2.4.インストーラをダウンロードする

対象のバージョンを選択し、ダウンロードします。

※本記事では2025版を選択

2.5.開発対象の言語とOSを選択し、ダウンロードする


　※本記事ではJava,Macを選択

2.6.ダウンロードしたファイルを開く

ダウンロードしたdmpファイルを開き、EclipseアイコンをApplicationsにドラッグ&ドロップします。


2.7.Eclipseを起動する

ApplicationsからEclipseを起動します。
警告が表示されたら、OKを選択します。
※警告が表示されない場合、2-9に進んでください。

2.8.アプリケーションの実行を許可する

システム環境設定　→　セキュリティとプライバシー　を開きます。
ダウンロードしたアプリケーションの実行許可　から、　このまま開く　を選択します。

その後、再度Eclipseを起動してください。

2.9.workspaceのディレクトリーを選択する

workspaceのディレクトリー選択し、起動してください。
ちなみにworkspaceとは、ソースコードのファイルなどを保管する作業場のようなものです。
ディレクトリーとはその作業場の配置場所のことです。本記事では初期設定のままで起動しておりますが、任意の場所を指定することもできます。

2.10.Eclipseが起動する

しばらく待つとeclipseが起動します。

お疲れ様でした！それでは、実際にさわってみましょう！

3.さわってみよう

3.1.Javaプロジェクトを新規作成する

左上のアイコンもしくはパッケージエクスプローラー内で右クリックからJavaプロジェクトを選択してください。


プロジェクト名を記入し、完了をクリックします。

パッケージエクスプローラー内に作成したJavaプロジェクトが表示されたら成功です。

3.2.クラスファイルを作成する

作成したプロジェクトフォルダを右クリックし、クラスを選択します。

クラスファイル名を入力し、public static void main(String[] args)にチェックを入れ、完了をクリックします。
この時点でチェックを入れることでmainメソッドの記述を短縮することができます。

プロジェクトフォルダ内にクラスファイルが作成され、エディタに自動生成されたメソッド・スタブが表示されます。

mainメソッド内(6行目)に以下のコードを入力します。
　System.out.println("Hello,World!");

アイコンもしくは⌘+S(macOS)もしくはctrl+S(windows)でファイルを保存します。

3.3.実行する

プロジェクトフォルダを右クリック→実行→Javaアプリケーション　をクリックします。

コンソール上にHello,World!と表示されたら成功です。お疲れ様でした！
